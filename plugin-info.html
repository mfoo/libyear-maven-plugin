Plugin Documentation

Goals available for this plugin:

Goal Description
libyear:analyze Analyze dependencies and calculate how old they are.
libyear:help Display help information on libyear-maven-plugin.
Call mvn libyear:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.6.3
JDK 11

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.mfoot</groupId>
          <artifactId>libyear-maven-plugin</artifactId>
          <version>0.0.1-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.mfoot</groupId>
        <artifactId>libyear-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
